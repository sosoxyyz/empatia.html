# empatia.html
Jogo de perguntas e respostas sobre empatia em que o jogador enfrenta situações do dia a dia e escolhe como agir. Cada escolha mostra consequências emocionais, ajudando a entender os sentimentos dos outros. O objetivo é desenvolver respeito, compreensão e solidariedade. No final, o jogador recebe uma pontuação de empatia.
